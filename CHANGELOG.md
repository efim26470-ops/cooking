# Offline Cookbook 4.9

- Полностью удалена подстановка случайных категорийных фотографий. Изображение пасты больше не может автоматически появиться у коктейля, десерта или салата.
- При первом запуске выполняется обязательная миграция старой базы: известные универсальные Unsplash-обложки и одна ссылка, назначенная разным блюдам, удаляются.
- Версия файлов в `index.html` синхронизирована с Service Worker. Ранее страница продолжала подключать `app.js?v=4.6.0`, из-за чего исправление 4.8 фактически не загружалось на iPhone.
- Сохраняются фотографии, которые пришли вместе с конкретным рецептом из TheMealDB, DummyJSON, Викиучебника или русской коллекции, а также пользовательские локальные фото.
- Для рецептов без фотографии добавлен строгий поиск по названию через русскую Википедию и Wikimedia Commons. Результат принимается только при достаточном совпадении ключевых слов.
- Одна найденная фотография не может быть присвоена двум рецептам с разными названиями.
- Если достоверную фотографию найти не удалось, показывается цветная обложка категории, а не случайное чужое блюдо.
- Добавлена рабочая команда `Ещё → Исправить фотографии`, которая повторно очищает повторы и запускает подбор для видимых карточек.
- Поиск фотографий ограничен двумя параллельными запросами и запускается только для карточек рядом с экраном, чтобы не перегружать Safari.
- Обновление интернет-каталога теперь проверяет новые фотографии до вывода карточек.
- Обновлены Service Worker и cache-busting всех ресурсов до 4.9.0.
