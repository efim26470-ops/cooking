# Offline Cookbook 4.6

- Полностью убран перехват внешних фотографий Service Worker: iOS больше не получает пустой закэшированный ответ вместо изображения.
- Добавлена ленивая загрузка только видимых карточек через IntersectionObserver.
- При блокировке исходного хоста выполняется одна попытка загрузки через оптимизирующий image-proxy.
- Цветная локальная обложка всегда остаётся под фотографией и видна при любой сетевой ошибке.
- Ошибка одной фотографии не ломает карточку и не запускает массовое удаление кэша.
- Обновлены версии файлов и Service Worker до 4.6.0.
