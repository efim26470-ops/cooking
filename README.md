# Offline Cookbook 2.0

Статическое PWA-приложение для iPhone, iPad, macOS и Windows. Сервер и собственная база данных не требуются: рецепты, фотографии, остатки, покупки и настройки хранятся на устройстве пользователя.

## Что входит в версию 2.0

- **199 подробных встроенных рецептов** на русском языке;
- пошаговые инструкции с отдельными этапами, ориентирами готовности и встроенными таймерами;
- описания блюд, сложность, примерная калорийность, советы, хранение и разогрев;
- исправленные обложки встроенных рецептов: больше нет пустого тёмного блока вместо изображения;
- корректное масштабирование ингредиентов с привычными дробями `¼`, `½`, `¾`;
- исправленная адаптация карточки рецепта для iPhone и широких экранов;
- автоматическое обновление старого встроенного каталога без потери избранного;
- очистка дублей интернет-рецептов;
- импорт **всех доступных рецептов**, а не прежнего ограничения в 140 записей;
- два интернет-источника: **TheMealDB** и **DummyJSON Recipes**;
- импорт фотографий, подробных инструкций, первоисточников, доступных видео, времени, порций, сложности и калорийности;
- локальное кэширование фотографий для дальнейшего офлайн-просмотра.

## Основные функции

- собственные рецепты и локальные фотографии;
- автоматический пересчёт ингредиентов по количеству порций;
- несколько параллельных кухонных таймеров;
- полноэкранный пошаговый режим с блокировкой сна экрана;
- список покупок с объединением одинаковых ингредиентов;
- учёт домашних остатков и сроков годности;
- подбор рецептов по доступным продуктам;
- категории для аэрогриля, духовки, мультиварки, плиты и блюд без техники;
- избранное, поиск, фильтры и темы оформления;
- импорт и экспорт всей книги в JSON;
- офлайн-оболочка через Service Worker.

## Интернет-каталог

Откройте `Ещё → Обновить интернет-каталог`.

Приложение последовательно:

1. получает полный доступный каталог TheMealDB по буквам;
2. добавляет коллекцию DummyJSON Recipes;
3. обновляет уже импортированные карточки без сброса избранного;
4. удаляет дубли;
5. сохраняет первые фотографии в локальный кэш сразу, остальные — в фоне.

Часть импортированных рецептов может оставаться на языке первоисточника. Ссылки на оригинальные страницы и видео сохраняются, когда источник их предоставляет.

## Публикация на GitHub Pages

1. Создайте новый репозиторий GitHub.
2. Загрузите **содержимое** папки проекта в корень репозитория.
3. Откройте `Settings → Pages`.
4. Выберите `Deploy from a branch`, ветку `main`, папку `/root`.
5. После публикации один раз откройте сайт с интернетом, чтобы Service Worker обновил локальный кэш.

## Установка на iPhone

1. Откройте опубликованный сайт в Safari.
2. Нажмите `Поделиться`.
3. Выберите `На экран Домой`.
4. Запустите установленное приложение.

## Хранение данных

Основные данные находятся в IndexedDB. Таймеры сохраняются в localStorage. Пользовательские фотографии уменьшаются и сохраняются в JPEG. Интернет-фотографии кэшируются через Cache Storage.

Очистка данных Safari или удаление данных сайта удалит локальную книгу. Периодически используйте экспорт JSON.

## Обновление старой версии

Замените файлы в репозитории новыми и дождитесь завершения GitHub Pages. После первого запуска приложение обновит встроенные карточки до версии 2.0 и сохранит пользовательские рецепты, покупки, остатки и избранное.
