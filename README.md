# Offline Cookbook 4.0

Статическое PWA-приложение для iPhone, iPad, macOS и Windows. Все пользовательские данные хранятся локально в IndexedDB. Проект готов к публикации через GitHub Pages.

## Новое в версии 4.0

- автоматическая фоновая загрузка интернет-каталога при первом запуске и далее раз в 7 дней;
- повторная синхронизация сразу после восстановления интернет-соединения;
- автоматический локальный перевод названий, ингредиентов, шагов, единиц измерения, кухни и подсказок на русский язык;
- сохранение оригинального названия и ссылки на первоисточник;
- четыре источника: TheMealDB, DummyJSON Recipes, TheCocktailDB и русскоязычный Викиучебник;
- исправленная классификация: завтраки, закуски, салаты, супы, основные блюда, овощные блюда, рыба и морепродукты, паста и лапша, гарниры, выпечка, десерты, соусы, напитки и заготовки;
- отдельные подборки: веганские, вегетарианские, до 400 ккал, белковые, без глютена, до 30 минут и напитки;
- автоматическая миграция уже загруженных англоязычных карточек и исправление их категорий;
- удаление дублей и обновление существующих карточек без потери избранного;
- локальное кэширование фотографий для офлайн-просмотра.

## Как работает автозагрузка

После запуска приложение проверяет дату последней синхронизации. Если каталог ещё не загружался, содержит мало интернет-рецептов или старше 7 дней, обновление запускается в фоне. В разделе `Ещё` можно отключить автообновление или запустить обновление вручную.

Перевод выполняется непосредственно в браузере без внешнего ключа API. Поэтому приложение остаётся полностью статическим и совместимым с GitHub Pages. Для редких названий используется транслитерация, оригинальное название всегда сохраняется.

## Публикация на GitHub Pages

1. Загрузите содержимое папки проекта в корень репозитория.
2. Откройте `Settings → Pages`.
3. Выберите `Deploy from a branch`, ветку `main`, папку `/root`.
4. После публикации откройте приложение с интернетом и дождитесь завершения первой фоновой синхронизации.

## Установка на iPhone

Откройте сайт в Safari → `Поделиться` → `На экран Домой`.

## Важно

Очистка данных Safari удалит локальную базу. Периодически сохраняйте экспорт JSON. Перед коммерческим использованием проверьте условия использования внешних источников и изображений.
